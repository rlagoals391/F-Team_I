# 🌊 Global Busan Guide: 데이터 기반 맞춤형 관광 큐레이션

> **"관광객이 아닌 현지인처럼, 데이터로 검증된 실패 없는 부산 여행"**

---

## 🔗 프로젝트 결과물
* **공식 웹사이트**: [Global Busan Guide 바로가기](https://copy-of-busan-real-discovery-555344686657.us-west1.run.app)

---

## 📌 프로젝트 개요
* **주제**: 유튜브 댓글 및 SNS 반응 분석을 통한 관광객 맞춤형 맛집/명소 큐레이션
* **목표**: 광고성 정보가 아닌 실 방문객의 생생한 데이터를 분석하여 외국인들이 겪는 언어 장벽과 메뉴 선택의 어려움을 해결하는 직관적인 정보 제공
* **주요 타겟**: 부산을 관광하는 외국인 관광객

---

## 🚀 주요 기능 및 특징
* **데이터 기반 큐레이션**: 구독자 10만 이상 해외 유튜버의 영상 댓글과 SNS(#BusanEats 등) 피드백 분석을 통해 신뢰도 높은 장소 추천
* **정서적 맥락 분석**: 단순 평점이 아닌 '매운맛 호불호', '오션뷰 선호도' 등 외국인 특화 감성 키워드 도출
* **맞춤형 테마 제공**: 데이터 분석 결과로 도출된 'K-Food 챌린지', '힐링 바다 산책' 등 인기 테마별 코스 안내
* **실패 없는 여행 경험**: 실제 방문객 데이터로 검증된 정보를 제공하여 낯선 도시 부산에 대한 안도감과 신뢰 형성

---

## 🛠 기술 스택 및 활용 도구
* **데이터 분석**: **Python** (유튜브/SNS 댓글 수집 및 시각화)
* **AI 활용**: 
    * **Google AI Studio**: 비정형 데이터의 감성 키워드 추출 및 해석 보조
    * 데이터 기반 스토리 구조 설계 및 문장 생성 보조
* **콘텐츠 제작**: 데이터 분석 기반 웹페이지, 스토리북, AI 홍보 영상 제작

---

## 📊 데이터 분석 인사이트
* **수집 대상**: 최근 1년간의 SNS 게시글, 해외 유튜버 영상 댓글, 주요 뉴스 기사
* **핵심 발견**:
    * 외국인 관광객은 '오션뷰'에 대해 매우 높은 긍정 감정을 나타냄
    * '매운맛'에 대한 호불호가 뚜렷하며, 구체적인 메뉴 정보(채식 옵션, 영어 메뉴판 등)에 대한 니즈가 높음
    * 단순 관광지 나열보다 현지인처럼 경험할 수 있는 실질적인 정보의 필요성 확인

---

## 🎬 프로젝트 스토리 (홍보 영상)
부산에 도착해 매운 음식과 복잡한 길 찾기로 당황하던 외국인 가족이 데이터 기반 가이드를 만나게 됩니다. 가족 취향에 딱 맞는 담백한 국밥집, 감천문화마을의 절경, 해운대의 조용한 쉼터를 추천받으며 부산의 진정한 매력을 느끼는 과정을 담았습니다.
